LESS动画优点

· 快速开发css3动画

· 采用less mixins写法，不会生成冗余css

· 已加入主流浏览器前缀，保证最大兼容性

· LESS-Animation 部分mixins支持传参，自定义动画幅度

（PS：sublime的less2css插件可能不支持编译本库的一些新写法，建议使用官方less编译、考拉、前端构建工具等方式编译）

使用方法

· 下载 lessAnimation.less 文件，git地址：

http://git.oschina.net/marchen/lessAnimate/blob/master/build/lessAnimate.less

· 在主less文件引入lessAnimation.less

@import "lessAnimation.less";

lessAnimation

定义引用

对于动画效果比较少的页面,可以直接在主less文件头部先定义需要用到的动画,然后后面需要用动画的地方,直接就可以引用了.(推荐给相同动画的不同效果命名不同的名字)

注意:不推荐这样引用动画,这样会生成多余的同名@keyframe定义

· 栗子一：引用 bounce 动画，只需在所在元素节点加入 .bounce();


    .bounce {
        .bounce();  //不推荐,如果是特殊定义的效果可以这样写.比如:
                    //.bounce(@t:100;@n:bounceHeight);
        animation: bounce 1s ease-in-out;
    }

推荐方式:


    //effect from lessAnimate.less
    .bounce();
    .bounce(@t:100;@n:bounceHeight);

    //Applying effect
    .bounce{
        animation: bounce 1s ease-in-out;
    }
    .flash {
        animation: bounceHeight 1s ease-in-out;
    }

· 栗子二: 如果是引用的效果比较多,可以新建一个pickAnimate.less来从lessAnimate.less中选择自己的动画,然后在主less文件中引用pickAnimate.less文件即可.

pickAnimate.less


    //pick animate effect from lessAnimate
    .bounce();
    .flash();
    .swing(@d:10deg,@n:swingHeight);

mixin传参引用

ps：
· 新增部分传参功能，可以自定义动画幅度参数，实现同类型动画不同幅度；
· 传参功能不影响原来上面直接引用，参数都可为空，原来的动画幅度已经设为默认值；

具体动画名和参数说明

动画名 参数说明 栗子
.bounce(@t; @n) @t：抖动最小幅度（默认值4px）；@n：动画帧名（默认值bounce） .bounce(5px; bounceA)
.pulse(@t; @n) @t：放大幅度（默认值1.05）；@n：动画帧名（默认值pulse） .pulse(1.5; pulseA)
.shake(@x; @n) @x：抖动最小幅度（默认值10px）；@n：动画帧名（默认值shake） .shake(20px; shakeA)
.swing(@d; @n) @d：旋转最小角度（默认值5deg）；@n：动画帧名（默认值swing） .swing(5deg; swingA)
.rotating(@d; @n) @d：旋转的角度（默认值360deg）；@n：动画帧名（默认值rotating） .rotating(180deg; rotateA); 注意.rotate(),在其他maxin库中是变形
.wobble(@d; @n) @d：摇摆最小幅度（默认值1deg）；@n：动画帧名（默认值wobble） .wobble(-2deg; wobbleA)
.fadeIn(@x; @y; @n) @x：x轴移动距离（默认值0）；@y：y轴移动距离（默认值0）；@n：动画帧名（默认值fadeIn） .fadeIn(-1000px; 0; fadeInA)
.fadeOut(@x; @y; @n) @x：x轴移动距离（默认值0）；@y：y轴移动距离（默认值0）；@n：动画帧名（默认值fadeOut） .fadeOut(-1000px; 0; fadeOutA)
.flipInDown(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipInDown） .flipInDown(700px; flipInDownA)
.flipInUp(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipInUp） .flipInUp(700px; flipInUpA)
.flipInLeft(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipInLeft） .flipInLeft(700px; flipInLeftA)
.flipInRight(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipInRight） .flipInRight(700px; flipInRightA)
.flipOutDown(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipOutDown） .flipOutDown(700px; flipOutDownA)
.flipOutUp(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipOutUp） .flipOutUp(700px; flipOutUpA)
.flipOutLeft(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipOutLeft） .flipOutLeft(700px; flipOutLeftA)
.flipOutRight(@p; @n) @p：perspective值，元素距视图的距离（默认值600px）；@n：动画帧名（默认值flipOutRight） .flipOutRight(700px; flipOutRightA)
